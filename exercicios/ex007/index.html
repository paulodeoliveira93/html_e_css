Imagens dinâmicas

Presta atençõa nas imagens

Veja como as imagens se ajustam de acordo com o tamaho da tela

Esse ajuste de tela automático é possivel porque em vez de colocarmos a tag img
colocamos a tag picture.
Dentro da tag picture temos que colocar 3 imagens diferentes:

  • Imagem grande
  • Imagem média
  • Imagem pequena
OBS: Esses tamanhos são opicionais, vc pode colocar quantas imagens quiser. Coloquei 3
para seguir um padrão de G, M e P.

Algumas observações sobre a tag picture

Devemos colocar a imagem do menor para o maior e a última imagem que vc for colocar use a tag img
Outra coisa que é importante deixar registrado aqui é que quando colocamos a tag picture
ela sugere outras 3 tag que vc precisar prestar atenção em alguns detalhes.

  1. media
    Aqui vc vai dizer o tamanho da imagem. Coloque max-width e o tamanho máximo da imagem
    Lembrando que deve colocar do menor para o maior.
  2. srcset
    Aqui vai ser a origem da imagem. Escolha a imagem. Da menor para a maior.
  3. type
    Aqui vc vai colocar: image/formado da imagen.
    Se o formato da imagem for png, coloca png. Se jpeg coloca jpeg e assim sucessivamente.

Áudio em html5

Agora vamos colocar áudios no html.

Desse jeito, colocamos uma audio no nosso codigo html

Alguns detalhes para ficar atento

A tag que usamos para colocar áudio é a tag audio.
Essa tag automáticamente preench outra tag que é a src, porém não vamos usar essa tag.
Apague essa tag e coloque a tag souce src type.

Mas uma coisa que devemos nos atentar é que não é todos os navegadores que carregam os áudios.
São 3 principais formados de áudio.

  • Formado em mp3 - mpeg
  • Formado em ogg - ogg
  • Formado em wav - wav

    • Voce precisa colocar esses 3 formados no seu código html
    Caso o navegador do usuário não consiga abrir o áudio, você deverá disponibilizar o áudio para baixar com a tag a.
